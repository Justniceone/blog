BlogDemo 2
==========

简介
----
BlogDemo2是一个博客系统，需求是参照了Yii1.0里面经典的例子blogdemo，然后用Yii2.0来进行改写的。



安装说明
--------

1.把文件解压后的目录blogdemo2拷贝到WWW服务器根目录。

2.建立数据库blogdemo2db，导入database文件夹中的SQL文件。

不同版本需要导入的SQL文件有不同，他们的对应关系参见database文件夹中的readme.md文件。

3.登录系统：
